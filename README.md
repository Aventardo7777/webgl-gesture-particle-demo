# WebGL 实时手势驱动三维粒子可视化渲染系统
### 🎮 在线演示地址：https://Aventardo7777.github.io/webgl-gesture-particle-demo/
> 点击链接即可在线体验手势交互粒子特效，无需本地下载代码

## 项目概述
本项目基于原生WebGL硬件加速渲染方案，结合浏览器媒体手部追踪API，构建一套低延迟、高流畅度的交互式三维粒子视觉引擎。
打破传统静态粒子特效局限，以实时手部姿态数据作为动态驱动源，实现手势动作与三维粒子集群的实时联动反馈，可广泛应用于交互大屏、数字艺术、虚拟交互展厅、线上视觉创意展示等场景。

## 核心技术优势
- 🚀 超低延迟手部实时捕捉与多节点手势姿态解析，毫秒级数据响应
- 🎨 WebGL底层硬件加速三维粒子渲染，海量粒子并行计算无卡顿
- ✨ 粒子集群动态参数实时映射手势：位移轨迹、尺寸缩放、渐变色彩、运动速度全维度联动
- 🧩 纯前端轻量化架构，零后端依赖、无第三方重型框架，单文件开箱即用
- 📱 兼容主流Chromium内核浏览器，适配PC端摄像头手势交互场景
- 🎭 高度可扩展粒子逻辑，支持自定义粒子运动曲线、特效叠加、交互触发规则

## 快速部署运行
1. 将仓库代码克隆至本地设备
2. 使用Chrome/Edge主流浏览器直接打开项目主文件 `3d-gesture-particle-system.html`
3. 授权浏览器摄像头权限，即可实时体验手势操控三维粒子视觉特效

## 底层技术架构
1. 三维渲染层：原生WebGL 标准着色器管线，自主封装粒子批量渲染器
2. 交互采集层：MediaDevices 媒体设备API + 手部关键点实时识别解析
3. 逻辑调度层：原生Vanilla JS 高性能帧循环调度，轻量化数据插值平滑算法
4. 视觉表现层：CSS3辅助视觉分层、粒子动态混合透明度、渐变色彩渲染算法
